Write your own distributed job scheduling framework using ETCD and Spring Boot 1 - Design

2020-07-01

In this tutorial we’ll build our own distributed job scheduler in Spring Boot with the help of ETCD, a distributed key-value store. To demonstrate the framework, jobs are distributively scheduled on worker nodes to fetch weather data of locations through REST calls and update an in-memory cache.
